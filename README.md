# 🌍 Um Brasil por Cidade

Projeto visual interativo que simula uma jornada pelo Brasil em 3D.

## 🚀 Tecnologias
- Three.js
- HTML / CSS / JavaScript

## ✨ Funcionalidades
- Globo 3D interativo
- Rota entre cidades
- Animação com scroll
- Pins de localização

## 🎯 Objetivo
Criar uma experiência visual estilo documentário viajando por todas as cidades do Brasil.
